## 🔧 Customização

### Alterar Número do WhatsApp
Procure por `5516994260416` e substitua pelo seu número (com DDI + DDD).

### Alterar Cores
Edite as variáveis CSS em `:root`:
```css
--neon-blue: #00D9FF;
--emerald: #10B981;
```

### Adicionar Analytics
Adicione seu código do Google Analytics ou Facebook Pixel antes do `</head>`.
