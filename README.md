# LeadExtract - Landing Page

## 📋 Descrição

Landing page profissional para o LeadExtract - Software de extração de leads do Google Maps.

## 🚀 Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- SigiloPay (Gateway de Pagamento)

## 🎯 Funcionalidades

- Landing page responsiva e moderna
- Sistema de checkout integrado
- Pagamento via PIX e Cartão
- Entrega automática por email
- Sistema de webhook para confirmação de pagamento

## 📦 Instalação

```bash
npm install
```

## 🚀 Desenvolvimento

```bash
# Frontend
npm run dev

# Backend API (Pagamentos)
npm run server

# Webhook Handler (Entrega)
npm run webhook
```
