## 🚀 Como Usar Agora

### Passo 1: Configure (2 min)
```bash
cd landing-page-v6
setup.bat  # Windows
# ou
./setup.sh  # Linux/Mac
```

### Passo 2: Teste (1 min)
Abra `index.html` no navegador e verifique tudo.

Escolha uma opção:
